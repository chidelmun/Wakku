# The Network

A Social Network Build with Node JS and the Jade Template Engine. 

# Getting started

### Installation
```bash
git clone https://github.com/chidelmun/Wakku.git
cd Wakku
npm install
```
### Setup database
In another window start MongoDB server
```bash
mongod
```
Seed the database 
```bash
make db
```

### Launch app
```bash
make start
```
or
```bash
node app
```

### Testing
```bash
make test
```
or
```bash
mocha test
```
